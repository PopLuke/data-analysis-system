# Data Analysis System

交互式数据分析与可视化系统初始项目。

## 技术栈

- Flask
- Pandas
- NumPy
- Scikit-learn
- Bootstrap

## 本地运行

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

浏览器打开：

```text
http://127.0.0.1:5000
```
